# system-calls-with-c
To implement the Process System Calls &amp; I/O System Calls

OS Required:
	Linux x64
	Linux x32
>> install a c compiler
>> Download the Repository

Open Terminal

>>cd system-calls-with-c
 
Once you're in that folder,

>>cd src

To Compile the source code

>>gcc systemCalls.c -o run

To execute the code
>>./run
